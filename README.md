# Los Déglingados

Site officiel de **Los Déglingados** — orchestre de bal populaire de Montpellier.
*Le Grand Bal des années 90 : le groupe qui s'amuse plus que vous !*

Site statique (HTML/CSS/JS, sans dépendance ni build) prêt pour GitHub Pages.

## Structure

- `index.html` — la page complète (styles et scripts inline)
- `assets/` — logo, photos de l'équipe, galerie, affiche de tournée, teaser vidéo

## Développement

Aucun outil requis — ouvrez `index.html` dans un navigateur, ou servez le dossier :

```bash
python3 -m http.server 8000
# puis http://localhost:8000
```
